Open Context Ontologies

This repository is for controlled vocabularies and ontologies used in Open Context. For the most part, the ontologies will be expressed in OWL. Current vocabularies / ontologies include:

  • Zooarchaeology Annotations

This is a work in progress. We welcome help and collaboration in authoring, fixing, and using these ontologies!
